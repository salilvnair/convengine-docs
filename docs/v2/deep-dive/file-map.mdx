## Agent Tool framework (`engine/agent/`)

ConvEngine's own DB-driven tool capability layer.

- `engine/agent/AgentPlanner.java` — LLM-based planning agent (CALL_TOOL / ANSWER loop)
- `engine/agent/AgentToolRegistry.java` — DB-backed tool registry (ce_agent_tool)
- `engine/agent/AgentConstants.java` — tool groups, actions, status constants
- `engine/agent/AgentDbExecutor.java` — SQL execution with template + guardrail
- `engine/agent/AgentSqlGuardrail.java` — allowlist-based SQL safety
- `engine/agent/AgentSqlTemplate.java` — SQL template renderer
- `engine/agent/model/AgentPlan.java` — planner output (action, tool_code, args, answer)
- `engine/agent/model/AgentObservation.java` — tool execution result
- `engine/agent/executor/AgentToolExecutor.java` — executor interface
- `engine/agent/executor/DbToolExecutor.java`
- `engine/agent/executor/HttpApiToolExecutor.java`
- `engine/agent/executor/FileToolExecutor.java`
- `engine/agent/executor/NotificationToolExecutor.java`
- `engine/agent/executor/DocumentRetrievalToolExecutor.java`
- `engine/agent/executor/WorkflowActionToolExecutor.java`
- `engine/agent/executor/CalculatorTransformToolExecutor.java`
- `engine/agent/executor/McpServerToolExecutor.java` — routes CALL_TOOL to external MCP servers
- `engine/agent/executor/adapter/*` — consumer adapter contracts
- `engine/agent/executor/handler/*` — DB query handler implementations
- `engine/agent/executor/http/*` — HTTP auth, request spec, response mapping
- `engine/agent/executor/interceptor/*` — query interceptors

## Real MCP protocol (`engine/mcp/`)

External MCP server connection, discovery, and invocation.

- `engine/mcp/McpRegistry.java` — server lifecycle, tool cache, `discoveredTools()`
- `engine/mcp/McpClient.java` — JSON-RPC 2.0 client (protocol version `2024-11-05`)
- `engine/mcp/McpTransport.java` — transport interface
- `engine/mcp/StdioMcpTransport.java` — subprocess stdin/stdout
- `engine/mcp/HttpMcpTransport.java` — HTTP POST transport
- `engine/mcp/SseMcpTransport.java` — Server-Sent Events transport
- `engine/mcp/McpServerConfig.java` — server config DTO
- `engine/mcp/McpDiscoveredTool.java` — runtime-discovered tool record
- `engine/mcp/McpController.java` — REST API for server management
- `engine/mcp/McpException.java`
