DISCONNECT_ELECTRICITY Example​

Turn 1
User
I need to disconnect electricity for account 10004567.
ConvEngine
Please share final service date and disconnection reason.

Engine asks follow-up because schema is incomplete (finalDate, reason missing).

Turn 2
User
Final date is 2026-03-31 and reason is moving out.
ConvEngine
Your request is submitted. Ticket DR-22991 is created.
  • Flow
  • Tables
  • SchemaExtractionStep.execute picks schema from ce_output_schema.
  • Prompt is resolved from ce_prompt_template (response_type=SCHEMA_JSON).
  • llm.generateJson(...) extracts fields and merges into context_json.
  • session.lockIntent("SCHEMA_INCOMPLETE") protects intent until required fields are complete.
  • RulesStep.execute can move state to submission-ready.
  • ResponseResolutionStep.execute returns clarification or success response.

Disconnect table touchpoints

TableExpected data
ce_output_schemarequired: accountId, finalDate, reason
ce_prompt_templateSCHEMA_JSON extraction template
ce_rulestate transitions for missing-fields and submit
ce_responseclarification + submission confirmation
ce_auditSCHEMA_EXTRACTION_*, SCHEMA_STATUS

LOG_ANALYSIS Example​

Turn 1
User
Analyze timeout logs for order flow.
ConvEngine
Please share correlation id or request id.

After identifier is collected, tool-assisted path can run.

Turn 2
User
Correlation id is 8fe6d9c9-2f7a.
ConvEngine
Root cause: DB pool saturation from 11:02 to 11:07 UTC.
  • Flow
  • Tables
  • SchemaExtractionStep extracts correlationId.
  • McpToolStep.execute plans tool call from MCP config.
  • Tool result is merged under context_json.mcp.
  • ResponseResolutionStep generates final RCA output.

REQUEST_TRACKER Example​

Single turn tracker
User
Track request REQ-778102.

Request REQ-778102 is IN_REVIEW and assigned to Queue-3.

  • Tables
  • IntentResolutionStep.execute resolves REQUEST_TRACKER.
  • SchemaExtractionStep.execute extracts request id.
  • RulesStep.execute sets tracking-ready state.
  • ResponseResolutionStep.resolveResponse selects mapping and emits output.
