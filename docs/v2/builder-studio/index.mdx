# Builder Studios

ConvEngine ships a visual builder environment that runs alongside the chat interface, accessible from the top navigation bar.

  title="Builder overview"

<Highlight type="info" title="Where it lives">
CE Builder Studio is a page inside <code>convengine-ui</code>. It is loaded via <code>PageContent</code> based on the <code>activePage</code> key set by the top navigation bar.
